Header Interaction - non-interactive Cards

There will be interaction states on the main header when there are actions directly attached on the card's header. There is no shadow elevation. This is only available on non-interactive Cards.
For interactive header use the .fd-card__header--interactive modifier with the .fd-card__header base class.
