Shadows create visual depth and hierarchy. These utilities and mixins offer predefined shadow levels for containers and elements, along with options for interaction and text shadows to improve clarity and user experience.

Helper mixins for shadow:

  • to set box-shadow: @include sap-set-box-shadow($value)
  • to set text-shadow: @include sap-set-text-shadow($value)
  • box-shadow: @include sap-shadow($type)
where:
$value can be a css variable or any value specified by the user. For example: @include sap-set-box-shadow(var(--sapContent_Shadow0))
$type can be one of the following: level-0 | level-1 | level-2 | level-3 | text | contrast-text | group-title-text | emphasized-text | header | shell | interaction | selected | negative | critical | positive | informative | neutral | none
